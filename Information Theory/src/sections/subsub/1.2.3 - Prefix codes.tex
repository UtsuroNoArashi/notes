    One of the most useful properties a code can have is that of being prefix-free;
        that is, no two codewords are prefix of one another.
        Thanks to this property, each codeword can be decoded unambigously,
        i.e. the word is decoded instantaneously.
        We call codes with this property \emph{prefix-free},
        or simply prefix. We also use the term instantaneous interchangeably.

    For this class of code the following results hold.
    \begin{theorem*}
        A code is instantaneous if and only if it's prefix-free.
    \end{theorem*}

    \begin{theorem*}
        All prefix-free codes are UD.
    \end{theorem*}
    \begin{proof}
        Follows as a consequence of Sardinas-Patterson theorem.
    \end{proof}
